# Project Description

This project is an IoT-Based Patient Vital Signs Monitoring System designed to improve healthcare monitoring.

The system uses sensors to measure heart rate and body temperature. These sensors are connected to a microcontroller, which processes the data and compares it with safe threshold values.

If abnormal values are detected, the system activates alerts such as a buzzer or LED. The system can also send data remotely using wireless communication.

The main goal is to provide continuous monitoring, early detection of health issues, and reduce the workload of healthcare workers.
