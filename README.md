# IoT-Based Patient Vital Signs Monitoring System

## Description
This project is an embedded system that monitors patient vital signs such as heart rate and body temperature using sensors and a microcontroller (Arduino/ESP32).

## Problem
Patient monitoring is often done manually, which can delay detection of health problems and increase workload for healthcare workers.

## Solution
This system continuously monitors vital signs, detects abnormal values, and sends alerts using a buzzer or LED. Data can also be sent remotely using IoT.

## Components
- Arduino / ESP32
- Heart rate sensor
- Temperature sensor
- WiFi/Bluetooth module
- Buzzer and LED

## How It Works
Sensors collect data → Microcontroller processes → System checks values → Alerts are triggered → Data sent remotely
